iPhone / iPad · Safari

  1. 1
    在 Safari 打开本页面（必须是 Safari 浏览器）。
  2. 2
    点击底部的 分享按钮（一个向上箭头的方框）。
  3. 3
    向下滑动并选择 「添加到主屏幕」。
  4. 4
    点击右上角 「添加」 完成。
提示：iOS PWA 仅在 Safari 中可安装；Chrome / 其他浏览器需先复制到 Safari。
🤖

Android · Chrome / Edge

  1. 1
    在 Chrome / Edge 中打开本页面。
  2. 2
    点击右上角菜单 ⋮，选择 「添加到主屏幕」 或「安装应用」。
  3. 3
    在弹窗中确认名称，点击 「添加」。
  4. 4
    主屏出现图标，点击即可像 App 一样启动。
💻

桌面 · Chrome / Edge

  1. 1
    地址栏右侧会出现 安装图标（电脑+向下箭头）。
  2. 2
    点击后在弹窗中确认 「安装」。
  3. 3
    完成后可从「开始菜单 / 启动器」独立启动。
小提示
安装后可离线浏览菜单、查看历史订单；新订单与商品更新需要联网。
