安装客户端

云栖浅食 · 健康轻食在线点餐

🚀 打开客户端 🔳 扫码访问
🍎

iPhone · Safari

  1. 1
    在 Safari 打开 客户端页面。
  2. 2
    点击底部 分享按钮 ⬆。
  3. 3
    向下滑动，选择 「添加到主屏幕」。
  4. 4
    点击右上角 「添加」。
🤖

Android · Chrome

  1. 1
    在 Chrome 打开 客户端页面。
  2. 2
    点击菜单 ⋮ → 「添加到主屏幕」 / 「安装应用」。
  3. 3
    确认添加，主屏会出现 🍜 图标。
安装后可离线浏览菜单、查看历史订单。
